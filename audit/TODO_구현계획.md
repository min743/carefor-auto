## 구현 방법 메모 (기술)

- 각 그룹 = `collector.py`에 페이지 이동(`build_spa_hash`로 SPA hash 생성) + 스크래핑 함수 1개.
- 검증된 패턴 재사용: XHR 훅(`anyXhrWait`)으로 팝업/그리드 응답 캡처, DOM은 `g-b`/`g-td` 플랫 구조.
- 판정 결과는 `analyzer`의 `item_results`에 `{"status": 양호/미흡, "detail": 건수 요약}` 형태로 추가
  → **대시보드·구글시트·요약페이지에 자동 반영**(추가 작업 불필요).
- 반기/분기/월 판정 헬퍼는 `analyzer.py`의 halfyear 로직 참고.
- 8-7·1-6·8-10 등의 정확한 SPA view 경로는 구현 시 케어포 URL 해시를 디코드해 확인.
- **34번은 예외적으로 수집이 이미 됨** — `audit_results/결과평가_<지점>/index.json`에 회차·작성일·총평이 있으니
  analyzer가 그 파일을 읽어 ①(반기 커버리지)·④(주1회 기록은 3-1 별도 필요) 판정하면 됨.
