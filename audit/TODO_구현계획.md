## 구현 방법 메모 (기술)

- 각 그룹 = collector.py 에 페이지 이동(build_spa_hash로 SPA hash 생성) + 스크래핑 함수 1개
- 오늘 검증된 패턴 재사용: XHR 훅(anyXhrWait)으로 팝업/그리드 응답 캡처, DOM은 g-b/g-td 플랫 구조
- 판정 결과는 analyzer 의 item_results 에 {"status": 양호/미흡, "detail": 건수 요약} 형태로 추가
  → 대시보드·구글시트·요약페이지에 자동 반영됨 (추가 작업 불필요)
- 반기/분기/월 판정 헬퍼는 analyzer.py 의 halfyear 로직 참고
- 8-7, 1-6, 8-10 등의 정확한 SPA view 경로는 구현 시 케어포에서 URL 해시를 디코드해 확인
