# 케어포 지점관리 점검 시스템 — 작업 요약 (2026-07-02~03)

새 채팅 세션 인수인계용. 코드 전체는 이 저장소(https://github.com/min743/carefor-auto)에 커밋되어 있음.

## 1. 시스템 개요

주간보호센터 4개 지점(둔산/서구/천안/청주 오창)의 **평가 매뉴얼 36개 항목 자동 점검** 시스템.
케어포(dn.carefor.co.kr)에 Playwright로 자동 로그인 → 1-1 수급자 정보관리에서 전 수급자(퇴소자 포함) 스캔
→ 판정 → 대시보드/구글시트/공개요약 갱신.

## 3. 대조 규칙 (전부 사용자 확정)

**기본 규칙** (어제 4개 지점 완료, 지점별 엑셀 존재):
- 낙상위험도 활동≥1 또는 걸음걸이≥1 → 욕구사정 일어나앉기/옮겨앉기 완전자립이면 불일치
- 둘 다 0점이면 완전자립이 정상

**추가 대조 6개 규칙** (2026-07-03 추가, 재스캔 후 make_extra_sheets.py로 시트 생성):
| # | 조건 | 불일치 판정 |
|---|---|---|
| R1 | 낙상 배변 ≥1점 | 화장실 사용하기 완전자립 ✗ |
| R2 | 욕창 영양상태 ≠4점 | 욕구사정 영양상태 "양호" ✗ |
| R3 | 욕창 움직임 ≤3점 | 일어나앉기·옮겨앉기 완전자립 ✗ |
| R4 | 낙상 정신상태 ≥1점 | 일어나앉기·옮겨앉기 완전자립 ✗ |
| R5 | 낙상 합계 ≥11점 | 옮겨앉기 완전자립 ✗ |
| R6 | 욕창 습기여부 ≠4점 | 화장실 사용하기 완전자립 ✗ |

주의: 욕창(Braden)은 점수 **높을수록 양호** (4점 만점) — 낙상과 반대.
수집 필드: falls{a활동, g걸음, bv배변, ms정신상태, total합계}, sores{항목별 score}, needs{sit, tr, toilet, nutrition}.
